## Portfolio Site: Ian Paulhamus ##

This portfolio site was developed using React and hosted using GitHub Pages. The site provides insight into who I am, my experience, my skills, and a collection of projects I have completed. 
Throughout the creation of this site, I followed the steps of design and preparation that a professional web designer would implement. Some of these steps include wireframing the layout and content, collecting assets and color palletes, then creating a detailed mockup of the site using canva.
These materials can all be viewed within this repository. This site displays proficiency in componetization in React alongside professional use of CSS styling and fundamental web design concepts.
